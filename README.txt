To compile any of the AStar programs go into the respective folder and:
javac AStar.java

To run any of the AStar programs go into the respective folder and:
java AStar

From there each AStar program asks for a starting city from the list of cities presented.


AStar1
This is the classic AStar algorithm without any danger or road conditions

AStar2
This algorithm uses danger in the hueristic function. The danger is added on to the cost of travel.

AStar3
This algorithm uses road conditions by subtracting the road condition from the hueristic funciton rewarding better roads.
